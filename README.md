# АИС учёта успеваемости — «Академия ТОП»

Автоматизированная информационная система учёта успеваемости (консоль Заместителя).
Реализован интеллектуальный модуль **«Генератор расписания»** с автораспределением,
ручной корректировкой и контролем коллизий, а также система авторизации.

## Запуск

Откройте `index.html` в браузере (Chrome / Яндекс.Браузер / Safari 12+).
Приложение самодостаточное — без сборки и сервера. Данные хранятся в `localStorage`.

## Возможности

### Генератор расписания (интеллектуальный модуль)
- **Матрица доступности преподавателя** (дни × пары): жёсткие запреты и мягкие предпочтения.
- **Алгоритм автогенерации** — рандомизированный жадный с multi-start (60 перезапусков):
  - жёсткие ограничения: запрет по матрице, нет двойной занятости, ≤4 пары/8 ак.часов в день на группу;
  - мягкие критерии: приоритет предпочтительных слотов, минимизация «окон».
- **Ручная корректировка** пар через drag-and-drop (с обменом местами).
- **Контроль коллизий** — подсветка конфликтов с указанием причины.

### Справочники
- Преподаватели (ФИО, программа РПО/КГИД, примечания, окна доступности).
- Дисциплины (тип, ак. часы, связка с преподавателями).
- Группы и учебная нагрузка.

### Авторизация и безопасность
- Закрытый вход (учётные записи только для Замов).
- Профиль: смена пароля, управление учётками, выход.
- Журнал действий (логирование, кто и когда что изменил).

## Технологии
Vanilla JS + CSS, шрифты Inter/Manrope. Без внешних зависимостей.
